# Project Management System

A full-stack project management application built with **FastAPI**, **PostgreSQL**, and **Next.js**.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | FastAPI (Python) |
| Database | PostgreSQL |
| ORM | SQLAlchemy |
| Migrations | Alembic |
| Auth | JWT (python-jose) |
| Password | bcrypt (passlib) |
| Frontend | Next.js 14 |
| Styling | Tailwind CSS |
| Charts | Recharts |
| HTTP Client | Axios |
| Testing | pytest |
